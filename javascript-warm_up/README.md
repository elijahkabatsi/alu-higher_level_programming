# JavaScript Warm-Up

This project introduces basic JavaScript concepts using Node.js.

## Requirements
- Ubuntu 20.04 LTS
- Node.js 14.x
- Editors: vi, vim, emacs
- All files executable
- `#!/usr/bin/node` as first line
- Semistandard compliant
- No use of `var`

## Usage
```bash
chmod +x *.js
./0-javascript_is_amazing.js
